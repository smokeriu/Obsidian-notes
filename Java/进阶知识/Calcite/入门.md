初次使用Calcite，需要如下行为：
1. 将Calcite依赖加入。
2. 编写自己的`ftl`文件和相关类。
3. 提取Calcite的`Parser.jj`模版。
4. 使用fmpp插件完成打包。

# 加入依赖
向maven中加入对应的依赖：
```xml
<dependency>  
    <groupId>org.apache.calcite</groupId>  
    <artifactId>calcite-core</artifactId>  
    <version>${calcite.version}</version>  
</dependency>
```
