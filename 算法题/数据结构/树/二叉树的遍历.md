# 递归遍历（DFS）

即先遍历左节点，然后遍历右节点，其模版如下：

```java
// 基本的二叉树节点
class TreeNode {
    int val;
    TreeNode left, right;
}

// 二叉树的遍历框架
void traverse(TreeNode root) {
    // 没有叶子了
    if (root == null) {
        return;
    }
    // 遍历左节点中所有子树
    traverse(root.left);
    // 遍历右节点中所有子树
    traverse(root.right);
}
```


上述代码可以用来构建二叉树的前/中/后遍历。即：

```java
void traverse(TreeNode root) {
    if (root == null) {
        return;
    }
    // 前序位置
    traverse(root.left);
    // 中序位置
    traverse(root.right);
    // 后序位置
}
```

在对应的位置处理root，即为对应的x序遍历。
