Java中提供的内置的数据结构：

# Java优先队列

> PriorityQueue

```java
PriorityQueue<Integer> pq = new PriorityQueue<>((a, b) -> a - b);
```

其中，`a-b`表示队列中元素如何比较，

# 栈

> Java中一般使用LinkedList来表示一个栈：Deque

```java
Deque<Object> stack = new LinkedList<>();
```

# 队列

> Java中一般使用LinkedList来表示一个队列：Queue

```java
Queue<Object> queue = new LinkedList<>();
```
