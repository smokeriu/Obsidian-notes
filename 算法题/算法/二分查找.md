二分查找主要用于在数组中，快速找到指定的元素，最主要的要求是数组**有序**。

其模版为：

> 假设为单调递增。

```java
while(right > left){
  int mid = (right - left) / 2 + left;
  // check

  if (mid > target){
	right = mid;
  } else {
    left = mid;
  }
}
```

二分查找最主要的是定义查找范围，即我们的更新后的新区间是否是闭区间。

# 闭区间

因为我们在check时，已经检查了midww vu

```java
while(right >= left){
  int mid = (right - left) / 2 + left;
  // check

  if (mid > target){
	right = mid - 1;
  } else {
    left = mid + 1;
  }
}
```

# 开区间

```java
while(right > left){
  int mid = (right - left) / 2 + left;
  // check

  if (mid > target){
	right = mid;
  } else {
    left = mid;
  }
}
```

# 计算mid值

计算mid值需要注意的是，最好采用如下方式：
```java
int mid = (right - left) / 2 + left;
```

这主要是避免`left+right`越界。
