# 递归遍历

即先遍历左节点，然后遍历右节点，其模版如下：

```java
// 基本的二叉树节点
class TreeNode {
    int val;
    TreeNode left, right;
}

// 二叉树的遍历框架
void traverse(TreeNode root) {
    if (root == null) {
        return;
    }
    // 遍历左节点中所有子树
    traverse(root.left);
    // 遍历右节点中所有zi uu
    traverse(root.right);
}
```
