
# 题目

- [15. 三数之和](https://leetcode.cn/problems/3sum/)
- [18. 四数之和](https://leetcode.cn/problems/4sum/)
# 思路

1. 先对数组进行排序。
2. 先确定第一个元素`a`，原则是：
	1. 为负数。因为已经排序过。
	2. 与上一个元素值不同，避免重复答案。
3. 通过左右双指针确定元素`(b,c)`。条件是：
	1. `a+b+c=0`。
4. 移动双指针，原则是：
	1. 与上一个元素值不同，避免重复答案。
	2. 双指针不交叉。

这题的要点是：
1. 数组中的一个元素只能被使用一次。但同一个值可以出现多次。
2. 一个三元组只能出现一次。所以通过排序来跳过重复的取值。
