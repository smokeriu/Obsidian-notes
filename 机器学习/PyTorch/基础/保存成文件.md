# 保存张量
可以将张量保存成文件：
```python
x = torch.arange(4)
torch.save(x, 'x-file')
```

并将其读取出来：
```python
x2 = torch.load('x-file')
x2
```


进一步，我们可以将多个张量一起保存：
## 通过列表
```python
y = torch.zeros(4)
torch.save([x, y],'x-files')
x2, y2 = torch.load('x-files')
(x2, y2)
```
## 通过Dict
```python
mydict = {'x': x, 'y': y}
torch.save(mydict, 'mydict')
mydict2 = torch.load('mydict')
mydict2
```
