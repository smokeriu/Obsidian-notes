通过`nn.Conv2d`的方式可以定义卷积层：
```python
nn.Conv2d(
		  in_channels: int,
		  out_channels: int,
		  kernel_size: _size_2_t,
		  bias: bool = True, ...
)
```
其中：
- `in_channels`：输入通道数。
- `out_channels`：输出通道数。
- `kernel_size`：卷积核大小。
- `bias`：是否含偏置。

> 关于通道，其理论可参考：[[../../深度学习/神经网络/卷积神经网络/卷积通道|卷积通道]]。

简单而言，卷积是一个2-4维的张量：
- `kernel_szie`：定义了卷积前两维的大小，所以其是一个二元组。
- `in_channels`：定义了卷积第三维的大小。
- `out_channels`：定义了卷积第四维的大小。
	- 如果设为1，则说明卷积
